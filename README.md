# Задание 1
Вы стажёр-продуктовый аналитик в МобиКар (сервисе посуточной и почасовой аренды автомобилей через мобильное приложение).

## Пользователи:

- Устанавливают приложение
- Проходят регистрацию
- Ищут авто рядом
- Бронируют и начинают поездку

## Проблема:

Конверсия пользователей с первого входа в приложение до первой поездки упала на 15% за последние 2 месяца.

## При этом:

- Установки выросли на 10%
- Удержание через 7 дней у зарегистрированных пользователей не изменилось
- Новых крупных обновлений не было, кроме изменений в поиске авто
- Отзывы в магазинах приложений не ухудшились

## Задачи:

### 1. Проанализируйте данные (за последние 8 недель)

[Данные.xlsx](Данные.xlsx)

### 2. Сделайте визуализацию данных

Создайте один график, который:

- Покажет динамику падения конверсии
- Поможет выявить, на каком этапе воронки происходит основной отток
- Визуально подчеркнёт ключевую проблему

**P.S.** Для создания графика можно использовать любой инструмент.
